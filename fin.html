Vous avez réussis à passer les deux épreuves de mathématiques et à suivre les QR codes afin de retrouver le proffesseur ainsi que les autres élèves ayant réussis l'énigme ! Ce projet géocache est maintenant terminé.


Charte Graphique :


Nous avons utilisé un code couleur afin de rappeler l'univers de l'école et de structuré notre texte. Des illustrations adaptées et différentes polices d'écriture ont également étés mobilisés.


Pour plus de détails :
  • Pour le fond : un vert foncé, dont le code HEX est #001501
  • Titres et texte : en blanc, de code HEX #00000
  • Mots importants : en jaune, de code HEX #e9ff00
Les polices utilisées sont :
  • Tahoma, sans-serif : pour les textes
  • Georgia, serif : pour les titres
  • Cursive : pour les titres secondaires
  • 'Courier New', monospace : pour les éléments mits en avant
  • fantasy : pour les messages d'encouragement
